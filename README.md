# 📖 Project Overview

This repository contains my completed work for the **Tata GenAI Powered Data Analytics Virtual Job Simulation** hosted on **Forage**.

The simulation focused on solving real-world business challenges faced by **Geldium Financial Services** using data analytics, predictive modeling, Generative AI, and Responsible AI principles.

Throughout this simulation, I performed:

- Exploratory Data Analysis (EDA)
- Predictive Model Planning
- Business Recommendation Development
- AI-Powered Collections Strategy Design
- Responsible AI & Ethical Risk Assessment
- Stakeholder Communication

---

# 🎯 Business Problem

Geldium Financial Services aims to reduce customer delinquency while maintaining positive customer relationships.

The challenge was to:

- Identify customers likely to become delinquent.
- Build a predictive modeling framework.
- Recommend business actions based on risk.
- Design an AI-powered debt collection strategy.
- Ensure fairness, transparency, and regulatory compliance.

# 📊 Project Tasks

## ✅ Task 1 – Exploratory Data Analysis (EDA)

### Objective

Analyze Geldium's customer dataset to identify:

- Missing values
- Data quality issues
- Customer behaviour
- Credit risk patterns

### Dataset Highlights

- 500 customer records
- Numerical and categorical features
- Credit utilization
- Credit score
- Income
- Loan balance
- Missed payments
- Debt-to-income ratio

### Key Findings

- High credit utilization strongly correlated with delinquency.
- Multiple missed payments significantly increased repayment risk.
- Missing values required preprocessing before modeling.
- Some high-income customers still showed poor credit scores, indicating potential anomalies.

### Skills Demonstrated

- Data Cleaning
- Data Interpretation
- Risk Analysis
- Exploratory Data Analysis

## ✅ Task 2 – Predictive Model Planning

### Objective

Design a predictive machine learning solution for identifying customers likely to become delinquent.

### Proposed Model

**Logistic Regression**

Chosen because it offers:

- High interpretability
- Probability-based predictions
- Regulatory transparency
- Business-friendly explanations

### Data Preparation

- Missing value imputation
- One-Hot Encoding
- Feature Scaling
- Feature Engineering
- Train-Test Split (80/20)

### Evaluation Metrics

- Precision
- Recall
- F1 Score
- ROC-AUC

### Responsible AI Measures

- Fairness Testing
- Bias Detection
- Data Privacy
- Human Oversight
- Continuous Monitoring

## ✅ Task 3 – Business Summary Report

### Predictive Insights

Major risk indicators included:

- Multiple missed payments
- High credit utilization
- High debt-to-income ratio

### Business Recommendation

Develop a proactive collections strategy by:

- Identifying high-risk customers early
- Sending personalized reminders
- Offering repayment assistance
- Improving customer engagement

### Ethical Considerations

The report addressed:

- Historical bias
- False positive predictions
- Human review before collection decisions
- Transparent AI explanations
